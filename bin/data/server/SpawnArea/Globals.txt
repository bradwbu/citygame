	/* Generator definition
	 *  Defines a type of entity generator.
 	 *  A user defined generator type name is expected after the keyword "Generator".
	 *  When a map is loaded, only generators with a corresponding
	 *  Generator definition will be active.
	 *
	 *  Recongized fields:
	 *	MaxSpawnCount
	 *		Defines how many entities the generator can spawn before
	 *		becoming inactive.  This field accepts an integer or the
	 *		string "infinite".
	 *		This field does not do anything currently.  The value
	 *		is assumed to be infinite.
	 *	MaxActiveEntities
	 *		Defines the maximum number of active entities the generator
	 *		can control.  When a generator reachs this limit, it must
	 *		wait until one its active entities to be destroyed before
	 *		it can spawn another.
	 *		This field accepts an integer.
	 *	ActivationInterval
	 *		Defines how often the generator is given a chance to spawn
	 *		an entity.  The interval should be specified in terms of
	 *		game ticks (currently at 30 ticks/sec).
	 *		This field accepts an integer.
	 *	GeneratedTypes
	 *		Defines the types of entities this type of generator can
	 *		spawn.  This field accepts a list of user defined entity
	 *		names and their spawning frequencies.
	 *
	 *		Note that the entity name must have already been defined
	 *		using a NPC definition or a Critter definition before it is
	 *		used here.
	 *
	 */
