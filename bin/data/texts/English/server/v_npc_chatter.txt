// You can specify which maps certain chat should be in by putting a
// [map ...] line in. You can list multiple maps if you want, just put them
// one after the other. All lines following the map lines will be heard only
// in those maps (until another [map ...] line occurs).
//
//    [map maps/City_Zones/City_01_01/City_01_01.txt]
//    [map maps/City_Zones/City_02_02/City_02_02.txt]
//    This is said only in 01_01 and 02_02.
//    [map maps/City_Zones/MonkeyVille/MonkeyVille.txt]
//    Ook! Only in MonkeyVille! Ook!
//
// To make subsequent chat lines show up in all maps:
//    [map all]
//
// You may also restrict the chat to only show up for heroes or villains.
// Preface the section with [hero] or [villain]. Chatter for everyone can
// be marked with [all].
//
//    [hero]
//    You are Heroic!
//    [villain]
//    You are Villainous!
//    [all]
//    You are something!
// 
// [map ...] resets the [hero] and [villain] settings for the lines
// that follow. Basically, after you specify a [map ...], you need to specify
// [hero] or [villain] again. If you don't, then the chat will be for [all].
//
// If there are no [map ...] lines, or if the map is set to [map all], then 
// the chat may appear in any map.
//
// If [hero] or [villain] is not specified, or if [all] is used, then anyone
// may get the chat.
